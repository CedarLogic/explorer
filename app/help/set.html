Definition
This tool builds a dummy variable according to the categories of the qualitative fields.
Parameters
In the left slot, the qualitative field whose categories are listed.
Several fields can be used successively to refine the partition.
Actions
Drag a qualitative field to the slot.
Click on the left buttons to remove records (observations) from the set being built.
Click on the right buttons to add records to the set.
Drag the arrow to the qualitative field list to create a new field with values YES and NO.
Drag the arrow to the quantitative field list to create a new numerical field with values 1 and 0.
