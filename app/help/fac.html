Definition
The analyses uses the contingency table of the two fields to compute line and column profiles, and considers the χ2 distance between them. It represents the profiles on the plane that maximizes the separability of the points.

Row profiles are displayed as red squares.
Column profiles are displayed as blue squares.
Parameters
In the top slot, the first categorical field (column profiles)
In the left slot, the second categorical field (row profiles)
Options
Option 1: The row and column profiles are displayed.
Option 2: Only the row profiles are displayed.
Option 3: Only the column profiles are displayed.
Actions
Enclosing some points (for example outliers) with the mouse will remove them from the analysis, they will be then placed at the bottom of the graph. Dragging them to the graph will add them back to the analysis.
